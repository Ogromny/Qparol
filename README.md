# Qparol
An encrypted password manager (NaCl, libsodium) in Qt with Ruby

### Installation
```sh
git clone https://github.com/Ogromny/Qparol.git
cd Qparol/
gem build qparol.gemspec
gem install parol-x.x.x.gem
```

###### OR

```sh
gem install qparol
```

### Execute
```sh
qparol
```
